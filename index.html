User Signup Dashboard

As a team project, we designed and developed a comprehensive user & opportunity sign-up dashboard using Looker Studio, tracking 27.562 sign-ups and 11.480 active participants across 106 countries while providing actionable insights on user activity patterns to support strategic decision-making.

As part of the team, I took full ownership of visualizing data and designing the interactive dashboard in Looker Studio. Collaborating closely with the team, I ensured the analysis aligned with strategic goals. Stakeholders had the ability to drill down into country-specific data and demographic insights, filter by year, gender, and other parameters to explore data in a personalized way. Tools used include Excel, Google Sheets, Looker Studio and PowerPoint.

Bike Sales Dashboard

This project showcases my expertise in Excel, including data cleaning, pivot tables, and dashboard creation. The dashboard provides insights into customer behaviour, income patterns, and commute distances, enabling non-technical stakeholders to make informed decisions.

I took full ownership of designing and developing an interactive dashboard in Excel. Starting with raw data, I performed data cleaning, created pivot tables, and designed a comprehensive dashboard to showcase key metrics such as average income per purchase, customer age brackets, and commute distances. The dashboard includes interactive slicers, enabling users to filter data by marital status, region, and education. This project demonstrates my ability to transform raw data into actionable insights, presenting a clear and engaging narrative for both technical and non-technical audiences.
